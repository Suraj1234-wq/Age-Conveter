# 🎂 Age Converter App

A simple and responsive Age Converter App built using React.js.  
This app calculates a user's age based on their date of birth.

---

## 🚀 Features

- Calculate exact age
- User-friendly interface
- Responsive design
- Instant result display
- Simple and clean UI

---

## 🛠️ Technologies Used

- React.js
- JavaScript
- CSS
- Bootstrap (optional)
